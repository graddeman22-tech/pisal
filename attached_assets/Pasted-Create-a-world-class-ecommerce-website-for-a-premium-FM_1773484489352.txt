Create a world-class ecommerce website for a premium FMCG brand named **PISAL**.

Domain: pisal.in

The website must be modern, extremely simple to use, fast, and visually attractive so that customers enjoy using it without confusion or burden.

The design should combine the best features used by the world’s top ecommerce websites such as Amazon, Apple Store, Nike, and Shopify stores.

The goal is to create a powerful ecommerce platform that looks like a premium brand and can handle many products in the future.

---

BRAND IDENTITY

Brand Name: PISAL

Theme Style:
Premium, clean, modern, trustworthy.

Primary Color:
Deep red

Secondary Colors:
Black, gold, white

Design style:
Minimal UI, modern typography, smooth animations.

---

LOGIN SYSTEM (IMPORTANT)

Users must be able to login using **mobile phone number with OTP verification**.

Login flow:

1. User enters phone number
2. System sends OTP to the phone number
3. User enters OTP
4. OTP verified
5. User account created automatically if first time user

Features:

Phone number authentication
OTP verification
Secure login session
Auto account creation
Logout system

OTP services that can be used:

Firebase Authentication
Twilio SMS
Fast2SMS API
